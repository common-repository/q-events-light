== Description ==

= Q Events Plugin Light =

Q Events is a new plugin to organize and publish all events on your website using Wordpress.

Your events can be planned and managed easily in the backend. In the front end all published events will be shown in a clear structured way. 


**Quick Facts**
The plugin has all you need to publish very simple events:
* event list in front end
* planend events
* published events
* archived events
* Event name / Location / Date / Ticket link / Ticket price


**Overview Back End**
On the back end a list of all published, planned and archived events is shown. Via one click a pop up will show up and a new event can be inserted. Same for updating an event.
The user can enter event name, subtitle, date, time, comment, location, city, picture via standard library, pre sales price and a ticket link. 
Same for update.
For every event you can add a link to your partner, who is coordinating the professional ticketing.
The toggle "Status" steers if an event is active, planned or archived.


**Overview Front End**
On the front end the events will be listed via a container in a same structure.
On the left side the picture will be shown and on the right side the ticket price + ticket link.

**Upgrade to pro**
You want to organize even more complex events like saving all details of your agencies, locations, partners, artists, shows: <a href="https://www.qanuk.io/plugins" target="_blank">Q Events Pro</a>

== Shortcodes ==
Via the shortcode [q_events] the plugin can be inserted in an own chosen site.


== Installation ==

The easiest way to start organizing and publishing your events is to login to your WordPress dashboard, go to Plugins > Add New, search for "Q Events Light", and click to install the plugin. You can also download the plugin zip file from this page and upload it from the Plugins > Add New > Upload page.

HOW TO: Add [q_events] on your preferred site. The active events will be shown on the site.
You can manage all events in the tab "Q Events" on your dashboard menu.

DOCUMENTATION: See <a href="https://www.qanuk.io/documentation/" target="_blank">Q Events Documentation</a>.


== Frequently Asked Questions ==

= Can I change the layout of the front end? =

Via CSS the layouts can be changed - be aware that an update might overwrite your changes.

= Do I need to fill all details? =

Only Event Name, date, location and city are required.

= Which language is available? =


The back end is available in English and German.
In the front end only your content will be shown.
